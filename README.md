# Описание страниц

- ## **Файл:** main.html  
  - **Для чего:** Оболочка приложения — шапка, меню и контейнер для подгружаемых страниц.  
  - **Читает из базы:** профиль пользователя (имя, аватар), настройки интерфейса (тема, язык), список выбранных предметов.  
  - **Сохраняет в базу:** изменения настроек, отметку о последнем визите (логин/активность).

- ## **Файл:** subjects.html  
  - **Для чего:** Страница выбора предметов — показывает все доступные предметы и даёт отметить свои.  
  - **Читает из базы:** перечень предметов (название, иконка, краткое описание).  
  - **Сохраняет в базу:** какие предметы пользователь выбрал (чтобы запомнить выбор).

- ## **Файл:** subject-config.html  
  - **Для чего:** Настроить каждый выбранный предмет — целевой балл и режим пробника.  
  - **Читает из базы:** название и иконку предмета, если есть — уже сохранённые настройки пользователя для этого предмета.  
  - **Сохраняет в базу:** целевой балл и выбранный режим (например, «не решать», «решать автоматически», «только разбор»).

- ## **Файл:** home.html  
  - **Для чего:** Главная страница после входа — плитки предметов, быстрый доступ к чату и пробникам.  
  - **Читает из базы:** выбранные предметы пользователя и их настройки, последние результаты пробников/тестов.  
  - **Сохраняет в базу:** быстрые пользовательские действия (например, закреплённый предмет, последнее открытое).

- ## **Файл:** chat.html  
  - **Для чего:** Большой полноэкранный чат с ИИ для разбора задач и помощи.  
  - **Читает из базы:** историю разговоров пользователя (если нужна), параметры сессий (какой предмет, режим).  
  - **Сохраняет в базу:** сессии чата и все сообщения (чтобы восстановить историю и проанализировать).

- ## **Файл:** tests.html  
  - **Для чего:** Список пробников/тестов, запуск теста и прохождение заданий.  
  - **Читает из базы:** наборы тестов и сами вопросы (текст, варианты, метки).  
  - **Сохраняет в базу:** начатые/завершённые попытки, ответы пользователя и полученные баллы.

- ## **Файл:** arena.html  
  - **Для чего:** Интерфейс для прохождения пробников в режиме «арены» (таймер, шаги).  
  - **Читает из базы:** текущий набор вопросов и конфигурацию теста.  
  - **Сохраняет в базу:** результаты попытки, время, детальные ответы.

- ## **Файл:** marathon.html  
  - **Для чего:** Длинные учебные циклы / марафоны — план занятий.  
  - **Читает из базы:** шаблон плана и элементы курса.  
  - **Сохраняет в базу:** прогресс пользователя по шагам марафона (что выполнено).

- ## **Файл:** roadmap.html  
  - **Для чего:** Дорожная карта обучения — визуальный план и цели.  
  - **Читает из базы:** шаблон дорожной карты и персональные цели пользователя.  
  - **Сохраняет в базу:** пользовательские цели и отметки выполнения.

- ## **Файл:** profile.html  
  - **Для чего:** Просмотр и редактирование профиля пользователя.  
  - **Читает из базы:** данные профиля (имя, email, аватар) и настройки.  
  - **Сохраняет в базу:** изменения профиля и персональные настройки.

- ## **Файл:** settings.html  
  - **Для чего:** Общие настройки приложения (уведомления, приватность, параметры ИИ).  
  - **Читает из базы:** текущие настройки пользователя.  
  - **Сохраняет в базу:** новые настройки пользователя.

- ## **Файл:** errors.html  
  - **Для чего:** Сообщения об ошибках / инструкция при сбоях.  
  - **Читает из базы:** обычно ничего критичного (можно показывать статичный текст).  
  - **Сохраняет в базу:** сведения об ошибках и падениях в систему аналитики (чтобы понять и исправить баги).
